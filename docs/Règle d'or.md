# 🏆 RÈGLES D'OR : Portfolio IA 2026

## 🎯 Philosophie : Business First, Technique Second

En 2026, un client ou recruteur ne cherche pas à savoir _comment_ c'est codé, mais _combien_ ça rapporte ou _combien_ de temps ça gagne.

### 1. La Hiérarchie de l'Impact

Chaque carte projet doit suivre cette structure :

- **Titre orienté bénéfice** (ex: "Facturation automatisée" au lieu de "Script PDF")
- **Une phrase d'explication** (Le "Quoi")
- **Impact Chiffré** (Le "ROI") 👉 Utiliser l'emoji 💥 et le vert néon `#00ffb4`

---

## 🏷️ Les 3 Étiquettes de Livraison (Stratégie MVP)

Pour les projets en cours ou partiels, ne jamais dire "Pas fini". Utiliser ces grades :

### 🟠 En cours de déploiement (Orange)

- **État :** Code stable, intégration serveur ou API en cours.
- **Message :** "Prêt à être allumé."
- **Couleur Code :** `orange-400` / `statusColor: 'orange'`

### 🔵 Version active (Bleu)

- **État :** Moteur fonctionnel, utilisé en interne ou en bêta restreinte.
- **Message :** "Le produit délivre déjà de la valeur."
- **Couleur Code :** `blue-400` / `statusColor: 'blue'`

### 🟢 MVP Validé (Vert)

- **État :** V1 complète, résultats prouvés et mesurés.
- **Message :** "Preuve de concept confirmée, prêt pour le scale."
- **Couleur Code :** `emerald-400` / `statusColor: 'green'`

---

## ✍️ Wording "Elite" (Lexique 2026)

- Ne pas dire "Dashboard", dire **"Pilotage"** ou **"Performance"**.
- Ne pas dire "Chatbot", dire **"Automatisation Support"**.
- Ne pas dire "En cours", dire **"Version Active"**.

---

## 🚀 Prochaine Étape : La Preuve Finale

Une fois la carte Home validée, la **Page Détail** doit contenir :

1. **Démonstration visuelle** (GIF ou Vidéo)
2. **Impact chiffré détaillé**
3. **Le bouton "Discuter de ce projet"** pour transformer le visiteur en prospect.
