# Lab version 2.5

Base de test isolée pour comparer une structure de portfolio plus simple et plus lisible.

## Contenu

- `index.html` homepage
- `projects.html` vue projets
- `styles.css` style partagé
- `app.js` animation légère

## Objectif

Tester une structure visuelle inspirée de la template fournie, tout en gardant la palette dark du projet.

## Usage

Ouvrir `index.html` dans le navigateur, puis comparer avec le portfolio principal avant toute intégration.
