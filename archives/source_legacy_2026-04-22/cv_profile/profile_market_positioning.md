# Profile Market Positioning

This document captures the part of the new resource that belongs to the profile and positioning track.

## Core message

The profile is not just "developer + AI". It is a practical profile for:

- automation
- business tools
- fast delivery
- AI-assisted workflow structuring

## Main positioning

Use the profile as:

> Developer who turns business needs into practical tools with Python and AI.

## What the market understands quickly

- Excel and file automation
- internal tools
- dashboards
- PDF and document generation
- data processing

## What should be emphasized

- Python
- React
- architecture
- delivery speed
- useful AI usage

## What should be avoided

- vague "AI expert" language
- overly technical marketing
- unclear positioning
