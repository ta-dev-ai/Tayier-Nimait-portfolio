# Prompts

This folder stores the standard prompt templates used by the Solena workflow.

## Standard prompts

0. load the project GPS and pipeline guide before any action
1. refine raw dialogues into a structured version
2. generate a Figma-quality dashboard image for the project
3. convert the approved image into implementation-ready code
4. turn the version into lab modules
5. turn validated modules into a project initialization plan
6. turn the methodology into a publication-ready article

## Rule

Each prompt must:

- have a clear role
- have a clear objective
- have explicit rules
- define input and output
- include an audit step
