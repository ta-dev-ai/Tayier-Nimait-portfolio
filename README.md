
# Bio-Inspired Portfolio v2.4

This repository is the clean base for the next version of my portfolio.

The goal of v2.4 is simple:

- keep the project free to deploy,
- use React as the public frontend,
- keep the content focused on hiring value,
- separate the new version from the current archive.

## Recrutement First

The current goal of this portfolio is not only visual quality. It must also help me get interviews in France by making my value easy to understand in a few seconds.

The target standard is documented here:

- [Portfolio Recrutement Standard](docs/portfolio_recrutement_standard.md)
- [Documentation Index](docs/INDEX.md)

## Technical Core
- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS with custom Glassmorphism and Neon utility classes.
- **Design Principles**: Dark-first, atomic components, and bio-responsive layouts.
- **Architecture**:
  - `domain/`: Business entities and logic.
  - `application/`: Use cases and services.
  - `presentation/`: UI components and page layouts.
  - `infrastructure/`: Static data and external integrations.

## Bio-Inspiration Philosophy
This portfolio uses biological systems as a metaphor for robust software engineering:
- **Resilience**: Fault-tolerant architectures.
- **Adaptability**: Component-based growth.
- **Efficiency**: Minimalist code footprint.

## Deployment
The public version is intended for free hosting, with GitHub Pages as the default target.
